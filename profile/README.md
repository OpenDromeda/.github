---

## What we build

### [MCC — Model Context Control](https://modelcontextcontrol.io)
The open governance layer for MCP (Model Context Protocol).

MCP defines how AI systems talk to tools. MCC defines **who gets to use MCP — and under what conditions.**

```
Policy enforcement · Audit logging · Rate limiting
```

→ [github.com/OpenDromeda/modelcontextcontrol](https://github.com/OpenDromeda/modelcontextcontrol)  
→ [modelcontextcontrol.io](https://modelcontextcontrol.io) — docs, diagrams, roadmap

---

## Philosophy

We believe the governance layer for AI tools should be:

- **Open** — source-available, auditable, forkable
- **Neutral** — no lock-in, no black boxes
- **European** — built with GDPR and data sovereignty as defaults, not afterthoughts

> *Who builds a server they can't control?*
